## Cách check tool

Xlua yêu cầu file **`@@`** để chạy dự án.  
Trong đó bạn cần khai báo các mục cơ bản:

-   **name**: tên dự án
-   **navigation**: nhóm & danh sách trang
-   **logo, favicon, colors**: giao diện
-   **api / openapi**: cấu hình API (nếu có)
-   **footerSocials, feedback**: tùy chọn thêm
